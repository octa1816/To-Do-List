# To-Do-List
📝 To-Do List Una aplicación simple de lista de tareas desarrollada con HTML, CSS y JavaScript. Permite agregar, marcar como completadas y eliminar tareas.  🚀 Funcionalidades Agregar nuevas tareas  Marcar tareas como completadas  Eliminar tareas  Guardado en localStorage (opcional)  🛠 Tecnologías HTML  CSS  JavaScript
