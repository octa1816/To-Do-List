📝 To-Do List
Una aplicación web simple pero funcional para gestionar tareas diarias. Diseñada con enfoque minimalista, desarrollada en HTML, CSS y JavaScript, ideal como proyecto de práctica para mejorar habilidades en desarrollo frontend.

🎯 Objetivo
Crear una lista de tareas donde el usuario pueda:

Añadir tareas pendientes

Marcar tareas como completadas

Eliminar tareas realizadas o innecesarias

(Opcional) Guardar las tareas en el navegador con localStorage
